# Thunderstore — Archived

The Thunderstore submission for `RyanEngineering/GlassPanel` was rejected ("Invalid submission")
and is not the active distribution path.

**Active distribution: NOMM / NOMNOM**
- Submit to https://github.com/KopterBuzz/NOMNOM via issue
- Users install via NOMM (https://github.com/Combat787/NOMM)
- Standalone: `install.ps1` or manual GitHub release download

Do not re-publish to Thunderstore without:
1. Resolving the rejection reason (likely `IPAddress.Any` + bidirectional socket control)
2. Getting explicit clearance from the Nuclear Option modding Discord
3. Confirming the panel works end-to-end first

The `LanMode = false` (localhost-only) default added in v1.9.0 addresses the main rejection
concern. If Thunderstore becomes relevant again, the code is ready — but NOMM is the right
target for this community.
